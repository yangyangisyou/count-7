# count 7

## O(n) version
If we iterative the number from 1 to n, we could easily count how many 7 within the range.

## How to test Sample test
```
npm run test
```

## Sample Input
1  
7  
20  
70  
100  
1000  

## Sample Output
g(1) = 0  
g(7) = 1  
g(20) = 2  
g(70) = 8  
g(100) = 19  
g(1000) = 271
